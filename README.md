# wordcloud
请结合 https://github.com/Quan666/ELF_RSS 一起使用
1. 将wordcloud.py放入src\plugins\XXXX\plugins目录下 xxxx可以是任何
2. 将wordcloud_bot放入data目录下
3. 运行 

    `pip install jieba`
    
    `pip install wordcloud`
